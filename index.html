Html2commonmark

Converts HTML to markdown using commonmark.js. Compliant with the commmonmark markdown specification.

Download this project as a .zip file Download this project as a tar.gz file

CommonMark is a rationalized specification of the Markdown syntax.

For more information about markdown and commonmark, see http://commonmark.org.
